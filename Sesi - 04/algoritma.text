Menghitung Luas dan Keliling Lingkaran

Note :
pi : 22/3
r : jari - jari
d : diameter (2r)

Rumus Luas Lingkaran : pi x r x r

Rumus Keliling Lingkaran : 2 x pi x r

Algoritma Luas Lingkaran :
1. Pilih Tab "Luas Lingkaran"
2. Input Jari - Jari
3. Input nilai pi
4. Klik button "Hasilkan"

Algoritma Keliling Lingkaran :
1. Pilih Tab "Keliling Lingkaran"
2. Input Jari - Jari
3. Input nilai pi
4. Klik button "Hasilkan"
